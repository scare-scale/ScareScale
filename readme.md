**Scare Scale** - A horror movie rating platform that scores films across five fear categories: Gore, Creepy, Jumpscares, Suspense, and Psychological.

## Features

- **5-Category Rating System** - Movies rated across Gore, Creepy, Jumpscares, Suspense, and Psychological categories
- **Scare Scale Approved** - Curated list of top-tier terrifying horror movies
- **Horror Rankings** - Browse rankings by category, year, and overall scare factor
- **AI Reviews** - AI-generated movie reviews analysing plot, characters, and scariness
- **User Reviews** - Submit and manage your own horror movie reviews
- **Search** - Find movies by title

## Getting Started

```bash
npm install    # Install dependencies
npm start      # Start development server
npm run build  # Build for production
npm run test   # Run tests
```
